#  Temperature Converter (Python)

##  Description

This project is a simple Python program that allows the user to convert temperatures between **Celsius** and **Fahrenheit**. It takes user input, applies the correct conversion formula, and displays the result rounded to one decimal place.

This project is great for Python beginners to practice:

* User input and output
* if / elif / else conditions
* Basic calculations
* String formatting

##  How the Program Works

1. The user chooses the temperature unit (`c` for Celsius or `f` for Fahrenheit)
2. The user enters the temperature value
3. The program calculates the correct conversion
4. The converted temperature is displayed

##  Example

Is this temperature Celsius or Fahrenheit (c/f): c
Enter the temperature: 25
The temperature in Fahrenheit is: 77.0°F
